# zh-main-journal

## 简介

zh-main-journal 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为多个 HTML 文档的集中存放与展示平台。

## 目录说明

```
/
├── index.html          # 仓库首页 / 页面索引
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源文件（CSS、JS、图片等）
    ├── style.css
    └── ...
```

- `index.html`：提供对 pages 目录下所有页面的简单导航或说明。
- `pages/`：各独立 HTML 页面文件，可自行组织子目录。
- `assets/`：页面共用的样式、脚本或媒体资源。

## 页面归档说明

- 每个 HTML 文件应保持独立、完整，不依赖外部不可控资源（建议内联或存放于 assets 目录）。
- 页面文件名建议使用英文小写、连字符分隔，避免特殊字符。
- 如需新增页面，直接在 `pages/` 下添加文件，并视情况更新首页索引。

## 维护说明

- 本仓库由维护者不定期更新，不保证所有页面长期可用或兼容最新浏览器。
- 提交新页面或修改时，请确保文件结构清晰，不引入外部恶意代码或跟踪脚本。
- 欢迎提交 Issue 或 Pull Request 讨论页面归档事宜，但请勿添加广告、营销或诱导点击内容。
- 仓库内容仅代表作者个人归档行为，与任何第三方组织或服务无关。
